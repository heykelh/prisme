# prisme
Plateforme d'audit de conformite AI Act pour les systemes IA de recrutement.  Les systemes IA de recrutement sont classes a haut risque par l'annexe III du reglement europeen sur l'IA. PRISME les audite contre les exigences des articles 9 a 15, sur la base de documents publics reels, avec des verdicts structures, sources et mesures.
